Заголовки

Заголовки являються одним из важнейших инструментов для структурировния текста

1.1 Рекоминдации по использованию

не Рекоминдуеться, чтобы на одной странице содержалось несколько заголовок верхнего уровня.

1.1.1 использование H1

Не рекомендуеться чобы на одной транице содержалось несколько заголовков верхнего уровня.

1.1.2 использование H2-H6

При использовании подзаголовок не Рекоминдуеться пропускать уровни заголовков, то есть после заголовка H1 должен идти подзаголовок H2 и только пото подзаголовок H3
